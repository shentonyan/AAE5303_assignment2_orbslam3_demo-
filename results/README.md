# Run Results

This directory contains results from running on MARS-LVIG AMtown02.

| File | Description |
|------|-------------|
| `CameraTrajectory.txt` | Full-frame estimated trajectory (TUM format) |
| `ground_truth.txt` | RTK ground truth extracted from bag |
| `metrics.json` | Evaluation metrics |
| `plots/` | Trajectory comparison plots |
